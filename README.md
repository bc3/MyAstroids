## Keys

⬆️ = speed up
⬇️ = speed down
⬅️ = turn left
➡️ = turn right

spacebar = shoot

b = spawn bullets powerup

s = span speed powerups
 
## Technical info

Monogame port of an old XNA game of mine

### Adding content

run command: dotnet mgcb-editor  
by doing this you can add items to your content lib, always set the correct type
